力学の基本

ここでの数学的記号などの説明

ここでは、物理量xの変化量をΔxとして表します。

時間や質量は向きがない大きさだけの量(スカラー)として考え、速度と運動量や力は向きと大きさのある量(ベクトル)として考えます。変数や定数を文字で表すときはスカラーは細字、ベクトルは太字で表記します。

また、yをxで微分するとは、 lim Δx→0 Δy Δxを求めること、つまり ∂y ∂xを求めることとします。

yをxで区間[a,b]の積分するとは、リーマン和 lim Δ→0 n Σ k=1 yiΔxiがinf lim Δ→0 n Σ k=1 y(i,sup)Δxi (ただし、y(i,sup)は[xi-1,xi]でyがとる値の上限)とsup lim Δ→0 n Σ k=1 y(i,inf)Δxi (ただし、y(i,inf)は[xi-1,xi]でyがとる値の下限)のいずれにも一致するときの値∫ b   a y dxを求めることとします。また、xが向きと大きさのある量(ベクトル)のときは、そのベクトルと同じ向きの単位ベクトルとベクトルの大きさと考え、単位ベクトルと被積分関数の積(ベクトルのときは内積、スカラーのときは単位ベクトルのスカラー倍)をベクトルの大きさで積分するものとします。

　　

ただし、リーマン和を求めるときのyiΔxi及びΔは、区間[a,b]をa=x0,b=xnとしてn個に分割し、Δxi=xi-xi-1としたときに、その中で最大のものをΔ、、yiは[xi-1,xi]でyがとる値のうちのいずれかと定義します。

また、今回扱うニュートン力学の問題では、通常は質量mについて ∂m ∂t = 0として考えますので、ここでもそうします。

力の定義

物体の質量、速度、運動量をそれぞれm,v,p としたときに、

p = mv

と表せます。

この式の両辺をtで微分すると、

∂ ∂tp = m∂∂tv

から、
∂ ∂tp = F, ∂∂tv = a
とすると、
F = ma

という式が成り立ちます。このとき、Fを物体に働く力、aを物体の加速度と定義します。また、この F = ma という式を運動方程式と呼びます。

また、F = 0のとき、Δp = 0であることから、Δv = 0が分かります。

力について

実は、重力や垂直抗力などその物体にかかる全ての力をFk(k = 1,2,…,n)としたときの合力 n Σ k=1 Fkが、Δpから求められる力です。
また、実際の物体には力が働いているので、Δp = 0である物体は、力がつりあいその物体に働く合力が0になっているのです。また、物体A,Bの位置ベクトルをそれぞれrA,rBとしたとき、物体Aに物体Bが及ぼす力Fは大きく分けると正の変数(定数を含む)kを用いて、F = k(rA-rB)と表せる引きつける力と、F = -k(rA-rB)と表せる遠ざける力に分けることができます。

引きつける力の例

  • 重力
  • 電荷の正負が異なる粒子同士に働くクーロン力

遠ざける力の例

  • 垂直抗力
  • 電荷の正負が等しい粒子同士に働くクーロン力

作用反作用の法則

n個の物体について考えるとき、それらの物体以外からの力が働いていないなら、それぞれの物体の運動量pk(k = 1,2,…,n)の総和 n Σ k=1 pkは変化しません。
これを運動量保存の法則と言います。

特に、2物体のときにはこの法則から2物体のそれぞれの運動量の変化量Δp1,Δp2はΔp1 = -Δp2ということが分かります。
このとき、それぞれの物体の運動量を時間で微分すると、
∂ ∂tp1 =  ∂ ∂t(-p2)
であり、
それぞれの物体に働く力をF1,F2とすると、
F1 = -F2がなりたちます。これを作用反作用の法則と言います。

また、2物体以外から力が働くとき、それぞれの物体に働く外部からの力(外力)の合計をそれぞれf1,f2とします。このとき、それぞれの物体に働く力は、それぞれ F1+f1,F2+f2であるからそれぞれの式をtで区間[t0,t1]の積分して運動量変化量Δp1,Δp2を求めると、
Δp1=∫ t1   t0 (F1+f1) dt ,Δp2=∫ t1   t0 (F2+f2) dtとなり、t0での運動量をそれぞれp1,p2とすると、それぞれの物体のt1での運動量は、
p1+∫ t1   t0 (F1+f1) dt, p2+ ∫ t1   t0 (F2+f2) dtとなります。
また、それぞれの物体が相互に力を加えない場合のt0からt1での運動量変化を考えると、
それぞれ ∫ t1   t0 f1 dt ,∫ t1   t0 f2 dtです。ここで、∫ t1   t0 f1 dtのような力を時間で積分した物理量を力積と呼びます。
なお、運動量の総和を変化させる要素は外力なので、
ここから外力による力積が存在し、運動量保存則が成り立たない状況でも、作用反作用の法則はなりたつことがわかります。

なので、今回とは逆に作用反作用の法則から運動量保存の法則をしめすこともできます。
